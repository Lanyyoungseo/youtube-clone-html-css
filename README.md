# YouTube Homepage Clone (HTML & CSS)

## Overview

This project is a practice exercise where I recreated the YouTube homepage layout using only HTML and CSS.

The goal was to improve my layout skills, especially:

* Flexbox
* CSS positioning
* Responsive design basics

---

## Tech Stack

* HTML5
* CSS3

---

## Features

* YouTube-style video grid layout
* Sidebar navigation
* Header with search bar
* Responsive structure (basic)
